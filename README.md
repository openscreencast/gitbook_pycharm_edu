# Gitbook - Pycharm Edu

Gitbook - Pycharm Edu

### Vorbereitung 

* NodeJS installieren
* Gitbook installieren: `npm install gitbook-cli -g`

### Anwendung

```
# Vorschau http://localhost:4000
cd book
gitbook install
gitbook serve

# Gitbook erstellen (statische Webseite)
cd book
gitbook install
gitbook build
```
